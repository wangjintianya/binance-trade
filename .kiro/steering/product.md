# Product Overview

Binance Auto-Trading System - An automated cryptocurrency trading application that integrates with the Binance exchange API to execute trades programmatically.

## Core Capabilities

- Secure API integration with Binance exchange using HMAC SHA256 authentication
- Real-time market data retrieval (prices, klines, account balances)
- Automated order management (market orders, limit orders)
- Risk control mechanisms (order limits, balance protection, rate limiting)
- Comprehensive logging and audit trail for all trading activities

## Target Users

Cryptocurrency traders who want to automate their trading strategies with built-in risk management and monitoring capabilities.
